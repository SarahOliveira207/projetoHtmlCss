Principais elementos de junção entre HTML e CSS

Para estilizar uma página HTML, indica-se o uso de um arquivo CSS externo. Há também a possibilidade de estilizá-lo em linha, porém não é uma boa prática no HTML 5

EM LINHA

<p style="color: ##027fe9;"> Exemplo de código </p>

EM ARQUIVO CSS


    #myCss{
        color: #027fe9;    
    }
